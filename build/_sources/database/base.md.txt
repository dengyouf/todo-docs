# MySQL 基础

## MYSQL部署管理
### MySQL安装

> [官网](https://dev.mysql.com/doc/refman/8.0/en/binary-installation.html)  : `https://dev.mysql.com/doc/refman/8.0/en/binary-installation.html`

前提条件
- 硬件：服务器标准化，根据业务场景和压力选择
- 软件：系统版本与参数优化
- 数据库版本标准化，推荐选择20以后的小版本
- 数据库软件工具标准化
